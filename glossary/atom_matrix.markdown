# Atom Matrix

M5Stackシリーズの中で最も小型な製品の一つである、手のひらサイズ（2.4cm四方）のESP32搭載マイコン開発ボード

## できること

[M5Stack ATOM Matrix 公式ページ](https://docs.m5stack.com/ja/core/ATOM%20Matrix)

- 5×5のLEDマトリクス：25個のカラーLEDを自由に光らせられる
- ボタン：押したことを検知してプログラムで反応できる
- 6軸モーションセンサー（MPU6886）：デバイスを傾けたり動かしたことを検知できる
- 赤外線LED：リモコンのような信号を送信できる
- Wi-Fi：ネットワーク通信が可能
